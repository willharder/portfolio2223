Project #5: Chemotaxis Project: Hot Air Balloon Festival!

About this Project:

This project was an attempt to resemble the Albequerque, New Mexico Hot Air Balloon festival, which is the largest festival in the world. My goal originally was to create an array of Hot Air Balloons that would randomly go across the screen, and then scale as they go farther away towards the mountains. My project then shifted once the scaler did not work, and I needed more randomness in the project. Then, I implemented mouseX and then the Balloon, in which could be changed by a balCount variable, and those Balloons then follow the mouse at different rates across the screen.

Project #1: Basketball Scoreboard!

About this Project:

This project was created following the 2021 AP Computer Science A Exam, for the last month of the school year. While when coding we were working in only a slightly different language, it felt like it was completely different. My original project didn't come out how I like, so I created this Basketball scoreboard. With a running clock, scores, fouls and giving half and full game statistics, it satisfies the basic needs of a scorekeeping device in Basketball. The clock is operated by the spacebar, and the scoreboard is increased using the numbers at the top of the keyboard. I was pleased with the output of the project!
